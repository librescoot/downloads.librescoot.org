Installer

Desktop app that walks you through installing LibreScoot on your scooter. Downloads firmware, flashes both boards, and sets up offline maps. All you need is a USB cable and a screwdriver.

Beta software. While we have tested on all platforms against real hardware, things can still go wrong. In the worst case, your scooter might become inoperable without deeper intervention. Use at your own risk. For support, try Discord, but keep in mind we're all volunteers.

Firmware Updates

To install an update, enable update mode via the display menu, then copy the .mender file over USB. For fresh installations, use the installer app above.

Map & Routing Data

Offline map tiles and turn-by-turn routing data, available per German state. Each one also covers about 20 km of the surrounding area. Download the map and routing files for your state and copy them to the scooter via USB. Only one state can be installed at a time. See offline navigation for setup instructions.
